# Screen Map

## 1. Inspection List

- List of apartments or acceptance sessions
- Shows address, building name, and draft progress

## 2. Plan Screen

- 2D floor plan image
- Overlay with tappable contour paths
- Zone types: wall, window, door, floor, ceiling, socket
- MVP source plan: `assets/plans/apartment_plan_mvp.jpeg`
- Mode switch: `2D` and `3D-like`
- `3D-like` is a visual preview mode, not the primary defect selection flow
- First annotated areas:
  - санузел
  - прихожая
  - спальня
  - кухня
  - гостиная
  - окно спальни
  - окно гостиной

## 3. Defect Report Screen

- Selected element summary
- Up to 3 photo slots
- Photo selection from iPhone photo library
- Direct capture from iPhone camera
- Local preview and removal before verification
- One text area for a shared comment
- CTA: send for verification

## 4. Verification Result Screen

- Verdict badge
- Confidence label
- Explanation
- Optional recommendation to retake photos or clarify text
